# Data Licensing and Sources

The data used in this project comes from multiple sources, each with its own terms of use and licensing. The source code of this project is licensed separately (see the `LICENSE` file).

This document outlines the licenses and terms applicable to the data.

---

### Bulbapedia

A significant portion of the data is sourced from Bulbapedia.

* **Source:** [Bulbapedia](https://bulbapedia.bulbagarden.net/)
* **License:** [Creative Commons Attribution-NonCommercial-ShareAlike 2.5 Generic (CC BY-NC-SA 2.5)](https://creativecommons.org/licenses/by-nc-sa/2.5/)

Under this license, you are free to share and adapt the data, provided you follow these terms:

* **Attribution (BY):** You must give appropriate credit to Bulbapedia as the source.
* **NonCommercial (NC):** You may **not** use the material for commercial purposes.
* **ShareAlike (SA):** If you remix, transform, or build upon the data, you must distribute your resulting work under the same CC BY-NC-SA 2.5 license.

---

### PokéAPI

Data is also retrieved from the PokéAPI.

* **Source:** [PokéAPI](https://pokeapi.co/)
* **License:** PokéAPI is a free and open-to-use service. While it does not specify a formal software license for its data, it operates under a "fair use" principle and requires attribution.

As requested by the PokéAPI creators, any project using the API should give credit to "PokéAPI and the community."

---

## Overall Project Constraint

Due to the **NonCommercial (NC)** clause in Bulbapedia's license, which is the most restrictive license governing the data, this project as a whole must be treated as non-commercial.

**Therefore, this project and any derivatives of it are strictly for personal, educational, and non-commercial use.**
